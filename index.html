Overview

This document contains coding standards for web sites and applications. It owes a great deal of debt to similar documents that have come before, including:

  • Roundarch Isobar Front-end Code Standards & Best Practices - Roundarch Isobar
  • Google HTML/CSS Style Guide - Google
  • The Essentials of Writing High Quality JavaScript - Stoyan Stefanov
  • jQuery Core Style Guidelines - jQuery
  • Code Conventions for the JavaScript Programming Language - Douglas Crockford
  • Principles of Writing Consistent, Idiomatic JavaScript - Rick Waldron
  • JavaScript Patterns by Stoyan Stefanov

Code consistency and adherence to best practices ensures readability, stability, and maintainability. These are the ideal to which we strive.
